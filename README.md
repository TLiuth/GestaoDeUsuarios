# This project aims to develop a simple CRUD web application.

## Functionalities:

- Registration page
- Login page
- User page
- User data redefinition
- Users management

## Technologies used

### Frontend

- TypeScript
- React
- Next.js
- Tailwind

### Backend

- Docker Compose
- Node.js
- NestJS
- TypeORM
- PostgreSQL
- Express.js (for Sessions)
